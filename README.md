## 🚀 Supported Runners

| Runner | OS |
|--------|----|
| `windows-latest` | Windows Server 2025 |
| `windows-2022` | Windows Server 2022 |
| `windows-11-arm` | Windows 11 ARM64 |
| `windows-11-vs2026-arm` | Windows 11 ARM64 + VS 2026 |

## 📦 Cách sử dụng

1. **Fork** repo này
2. Vào **Actions** > **AcherLab RDP** > **Run workflow**
3. Chọn thời gian (1h / 3h / 5h40m)
4. Chờ vài phút, nhận thông tin kết nối

## 🔐 Thông tin đăng nhập

- **User:** `AcherLab`
- **Pass:** `AcherLab@2026!`
- **Port:** `3389`

## 📋 Yêu cầu

- GitHub Account (free)
- Mỗi runner chạy độc lập, không ảnh hưởng lẫn nhau

## ⚠️ Lưu ý

- Thời gian tối đa: 5h40m (giới hạn GitHub Actions)
- Các runner chạy song song, mỗi runner là một máy riêng biệt
- Tự động dọn dẹp khi kết thúc
